# grpc-playground
GRPC Pojects built using Java and Spring Boot


Protocol Buffers:

- no type safety
- data size is larger
- also known as Protobuf/Proto
- developed by google
- platform/language neutral for serialising and deserialising 
- it is a binary format
	- not human friendly
	- takes less time to serialise and deserialise
	- size is smaller compared to json
- very fast for inter-service communication
